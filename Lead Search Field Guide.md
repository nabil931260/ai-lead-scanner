# Lead Search Field Guide

Source: [[Method 2 Website Analyzer Playbook]]

Use this note while manually finding leads to add to [[Leads Input]].

## Current Goal

Find public business websites with visible quote, booking, intake, or follow-up friction.

The target is not "any local business." The target is a business where a short audit could plausibly help them stop losing leads or make follow-up easier.

## Best Search Pattern

Use this formula:

`[niche] [city] [quote/contact/estimate/service]`

Examples:

- `cleaning service Plano quote`
- `commercial cleaning Richardson contact`
- `painters Plano free estimate`
- `remodeling Richardson contact form`
- `plumber Plano emergency estimate`
- `HVAC Frisco request service`
- `mobile detailing Plano book appointment`
- `catering Dallas event inquiry`
- `tutoring Plano contact`
- `personal trainer Richardson consultation`

## Best Cities To Start

Use nearby/local areas first:

- Plano
- Richardson
- Frisco
- Dallas
- Garland
- Allen
- McKinney
- Carrollton

Start with one city per batch. Mixing too many locations makes comparison harder.

## What To Search For On Each Site

Open the website and check:

- homepage
- contact page
- quote / estimate page
- booking / appointment page
- services page
- FAQ page if available
- footer contact info

Look for:

- "request a quote"
- "free estimate"
- "schedule service"
- "call/text us"
- "we will get back to you"
- contact form with very little detail
- long service menus with no clear next step
- recurring service language
- emergency/urgent service language
- multiple contact paths with no clear tracker
- no FAQ
- no obvious confirmation/follow-up process

## Good Lead Signals

Strong:

- contact or quote form exists but next step is unclear
- business asks customers to call, text, email, or fill a generic form
- many services/packages require custom quotes
- recurring customers/jobs are likely
- public reviews mention response time, scheduling, or communication
- website is functional but messy enough to audit

Medium:

- site has basic contact info but no quote flow
- service area and services are clear, but follow-up process is not visible
- business seems active but website copy is thin
- booking is partly manual

Weak:

- polished booking system already exists
- national/corporate site controls the local page
- no clear intake problem
- no public contact path
- business is too regulated or sensitive for a starter offer

## Quick Lead Score Before Adding

Score manually before adding the lead:

| Question | Points |
|---|---:|
| Do they likely depend on quotes/bookings/inquiries? | 2 |
| Is there a public website URL? | 2 |
| Is the contact/quote path visible? | 2 |
| Is there a clear follow-up or tracker pain hypothesis? | 2 |
| Is the business small/local enough to care? | 1 |
| Can I write a specific first sentence? | 1 |

Add leads scoring 7+.

Maybe add 5-6 if the niche is very promising.

Skip below 5.

## One-Sentence Pain Hypothesis Formula

Use this in the `Notes` field:

`[Visible signal], so [likely workflow/follow-up issue].`

Examples:

- `Free-estimate form asks for project details, so quote follow-up may be tracked manually.`
- `Recurring cleaning services are listed, so client schedule and follow-up tracking may matter.`
- `Emergency service language pushes calls and forms, so missed-call and quote tracking may be painful.`
- `Event inquiry page asks for custom details, so availability and deposit follow-up may be scattered.`

## Niche-Specific Angles

| Niche | Best visible signal | Offer angle |
|---|---|---|
| Cleaning services | quote/recurring service requests | quote tracker + recurring client follow-up |
| Painters/remodelers | free estimates/project details | estimate intake + follow-up tracker |
| Plumbers/HVAC | urgent service/contact forms | missed inquiry + service request tracker |
| Mobile detailers | text/book/package language | booking intake + repeat customer tracker |
| Tutors/coaches | consultation/intake language | student/client inquiry tracker |
| Photographers | package/availability inquiries | inquiry-to-booking pipeline |
| Caterers/event vendors | event inquiry and availability | event quote and deposit follow-up tracker |

## What Not To Add

Do not add:

- social media profile URLs as websites
- LinkedIn, Instagram, Facebook, TikTok, or login-gated URLs
- personal emails from private profiles
- scraped contact lists
- businesses where the website blocks public review
- leads where you cannot name a visible signal
- duplicates unless the location/contact path is clearly different

## Search Session Checklist

Before searching:

- Pick one niche.
- Pick one city.
- Decide target count: 5 or 10.

For each candidate:

- Open website.
- Check contact/quote path.
- Write one pain hypothesis.
- Find public contact path.
- Add only if it scores 7+.

After the batch:

- Run Method 2 scripts.
- Review top scored leads.
- Pick 3-5 for outreach.
- Stop searching and send reviewed messages.
